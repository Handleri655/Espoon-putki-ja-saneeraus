# Espoon Putki & Saneeraus Oy – Nettisivut

Modernit staattiset nettisivut putki- ja saneerausyritykselle.

## Käynnistys

Avaa `index.html` suoraan selaimessa tai käytä paikallista palvelinta:

```bash
npx serve .
```

Sivut ovat saatavilla osoitteessa `http://localhost:3000`.

## Rakenne

```
├── index.html      # Pääsivu
├── css/style.css   # Tyylit
└── js/main.js      # Navigaatio, animaatiot, lomake
```

## Ominaisuudet

- Responsiivinen design (mobiili, tabletti, työpöytä)
- 24/7 päivystyspalkki kiinteänä yläreunassa
- Palveluosiot, hinnoittelu ja yhteystiedot
- Yhteydenottolomake (avaa sähköpostiohjelman)
- Scroll-animaatiot ja mobiilivalikko

## Julkaisu

Lataa tiedostot mihin tahansa staattiseen hosting-palveluun (esim. Netlify, GitHub Pages, oma palvelin).
